# Smslane

A simple wrapper gem for the smslane.com HTTP API for sending Bulk SMS in India.

## Installation

Add this line to your application's Gemfile:

    gem 'smslane'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install smslane
